# taskaroo-manager

Create a task management app like Asana.

Every task has field - title, description, checklist, due date, priority and an opion to create as many custom fields per task.

The view of all tasks is a list view which can be swithced easily between a kanban view and a calendar view

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/taskaroo-manager.git
cd taskaroo-manager
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
